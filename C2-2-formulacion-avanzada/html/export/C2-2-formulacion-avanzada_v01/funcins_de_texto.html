Funcións de texto

En moitas ocasión imos necesitar operar con textos, por exemplo para unir uns textos con outros, ou para descompoñer un código de texto en diferentes partes, ou realizar conversións entre maiúsculas e minúsculas, entre outras. Seguidamente imos detallar algunhas das que consideramos máis utilzadas polos usuarios. Recomendamos, coma sempre, revisar mediante o asistente para descubrir outras funcións dispoñibles.

Función CONCATENAR()

CONCATENAR une textos.

Sintaxe básica:

=CONCATENAR(Texto1;[Texto2];...)

Onde Texto1, Texto2, etc... son textos ou referencias a celas con textos.

Como alternativa á función CONCATENAR podemos utilizar o operador & de concatenación.

Sintaxe básica:

=Texto1&[Texto2]&...

O resultado final é idéntico.

Recorda que os textos dentro das fórmulas deben ir sempre entre comiñas, por iso se introduciu un espazo para separar o nomoe do apelido no resultado amosado.

Función MAIÚSCULA()

MAIÚSCULA converte un texto a maiúsculas.

Sintaxe básica:

=MAIÚSCULA(Texto)

  Nesta versión de Calc o nome da función escríbese con acento.

 

Función MINÚSCULA()

MINÚSCULA conveerte un texto a minúsculas.  

Sintaxe básica:

=MINÚSCULA(Texto)

 

Función PRIMMAIÚSCULA()

PRIMMAIÚSCULA converte a maiúscula a primeira letra de cada palabra dentro dun texto.  

Sintaxe básica:

=PRIMMAIÚSCULA(Texto)

 

Función LONX()

LONX obtén a lonxitude en caracteres dunha cadena de texto.  

Sintaxe básica:

=LONX(Texto)

Función BUSCAR()

BUSCAR localiza a posición dun texto dentro doutro; a busca é indiferente a maiúsculas ou minúsculas.

Sintaxe básica:

=BUSCAR(Texto_buscado;Texto_onde_buscar;[Posición_inicial]

onde

  • Texto_buscado é o texto que se desexa localizar dentro de Texto_onde_buscar
  • Posición_inicial é un argumento opcional para indicar a partir de que posición de Texto_onde_buscar se desexa efectuar a busca.

No seguinte exemplo, a función BUSCAR atopa a posición número 8, correspondente ao 'e' de 'rematou' (aínda que a busca se facía dun 'E' maiúsculo).

Neste outro exemplo, repítese a busca pero a partir da posición número 9, polo que a función devolve o resultado 17 que corresponde á posición do 'E' de 'Entroido'.

Se falla a busca (ao non atopar ningún 'E' máis alá da posición 18), devolver o erro #VALOR.

Función ATOPAR()

ATOPAR localiza a posición dun texto dentro doutro; a busca é sensible a maiúsculas e minúsculas.  

Sintaxe básica:

=ATOPAR(Texto_buscado;Texto_onde_buscar;[Posición_inicial]

onde

  • Texto_buscado é o texto que se desexa localizar dentro de Texto_onde_buscar
  • Posición_inicial é un argumento opcional para indicar a partir de que posición de Texto_onde_buscar se desexa efectuar a busca.

Neste exemplo, como a busca se está facendo sobre o 'E' en maiúsculas, ATOPAR salta por alto a coincidencia do 'e' minúsculo de 'rematou' e devolve a posición do 'E' maiúsculo de 'Entroido'.

Función ESQUERDA()

ESQUERDA obtén un determinado número de caracteres pola esquerda dunha cadea de texto.  

Sintaxe básica:

=ESQUERDA(Texto;Número_caracteres)

Neste exemplo, extraense 11 caracteres desde a esquerda do texto de referencia.

Función DEREITA()

DEREITA obtén un determinado número de caracteres pola dereita dunha cadea de texto.  

Sintaxe básica:

=DEREITA(Texto;Número_caracteres)

Neste exemplo, extraense 6 caracteres desde a dereita do texto de referencia.

Función MEDIO()

MEDIO obtén un determinado número de caracteres dunha cadea de caracteres a partir dunha posición inicial.

Sintaxe básica:

=MEDIO(Texto;Posición_inicial;Número_caracteres)

Neste exemplo, obteñense 6 caracteres a partir da posición 7 do texto de referencia. Nótese que a cadea de texto devolta inclúe un espazo inapreciable, que sería o sexto carácter da cadea obtida

Función RECORTAR()

Nesta función, equivalente á función ESPAZOS doutros programas, elimina os espazos iniciais, finais e deixa tan só un espazo de separación entre palabras se hai máis de un.  

RECORTAR elimina espazos sobrantes dun texto.

Sintaxe básica:

=RECORTAR(Texto)

 

 

No exemplo hai espazos adicionais antes, despois e dentro do texto. Foi subliñado o texto para que se visualicen os espazos adicionais incluídos.

Amósase o resultado de LONX como comprobación, antes e despois de aplicar a función RECORTAR.

Función TEXTO()

Esta función permite convertir un valor numérico nun texto representando o mesmo valor cun formato determinado. Resultan moi útiles naqueles casos nos que non chega con cambiar o formato dun número para obter a información desexada.

Por exemplo, poderíamos mostrar unha data nun formato que só amose o nome do mes, pero realmente a cela seguiría contendo a data. Usando esta función obteremos realmente un texto co nome do mes. Tamén pode ser moi útil cando necesitemos concatenar textos con números.

TEXTO converte un valor numérico nun texto que representa o valor nun formato dado.  

Sintaxe básica

=TEXTO(Valor;Código_formato)

 

onde Valor é un importe numérico e Código_formato un dos formatos que se amosan no cadro Código de formato na lapela Números do cadro de diálogo Formato celas cando se aplica un formato de número.

 Advertencia
Se desexamos inserir un texto literal debemos preceder cada carácter cunha barra invertida  \  (código de escape) para que o formato non o interprete como un código.
